## Installation
<br>

- Clone the repository using:

```
git clone git@github.com:RahulPanwar1/Todo-list.git
```

- To Ensure the backend run properly
- Start the MongoDB server
- Then in terminal
```
node app.js
```
- Go to localhost:3000
